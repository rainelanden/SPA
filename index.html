Internets uppkomst

Internet har en föregångare som heter ARPANET och det skapades under kalla kriget av det amerikanska försvarets utvecklingsavdelning DARPA. Det var färdigt år 1969. ARPANET var till för att användas av militären, så att de enkelt kunde kommunicera med varandra. Underlättningen var stor då de inte hade något kommunikations-verktyg förut. Det var inte menat att användas av allmänheten så som det används idag med det moderna internet som vi känner till. Det började med att de senare vid år 1971 skickade e-post mellan 2 st datorer som stod bredvid varandra. Det är på grund av Internet som vi kan surfa, chatta och skicka meddelande så som e-post. År 1989 så stängde man ner ARPANET och Internet skapades. Även Internet var väldigt begränsat för användning då det endast var tillgängligt för forskare och militär.

World Wide Web

Det var Tim Berners-Lee som år 1990 skapade webben (World Wide Web). Webben är en internettjänst som gör det möjligt att datafiler och t.ex. websidor blir tillgängliga. Webben består av 3 olika saker: URL, HTTP och HTML. URL bestämmer hur sidan ska se ut. HTTP bestämmer hur webbsidan ska skickas. HTML bestämmer hur innehållet på webbsidan ska visas. 1991 så kom den första webbplatsen att bli tillgänglig för alla. För att få tillgång till webben måste man ha en webbläsare för att kunna söka efter information. Olika exempel på webbläsare som används idag är: Google Chrome, Safari, Internet Explorer, Mozilla Firefox. En av de första som kom var Netscape och den användes flitigt under 90-talet då den var väldigt dominerande. Internet Explorer var ledande under sent 90-tal/tidigt 2000-tal men nu under senare år har Google Chrome och Mozilla Firefox tagit över marknaden ganska radikalt. Webben var revolutionerande eftersom att det redan samma år som det kom fanns ungefär 200 st registrerade WWW-servrar. 2 år senare så ökade antalet rejält, det fanns då 30 000 WWW-servrar. Det ökar för varje år som går.

Hur påverkas samhället?

Webben och internet har definitivt ändrat hur vi lever då det påverkar våra liv något enormt. Internet har givit oss människor ett helt nytt sätt att utbyta information och att konversera med andra människor runt om i världen. Från början så skickade människor endast mejl genom internet, men nu är det andra tider. Efter att internet blossade upp så används det till allt möjligt. Nu för tiden används internet till att söka upp information som aldrig förr. I princip all information finns tillgänglig att leta upp. Samhället hade säkerligen sett väldigt annorlunda ut om det inte var för internets revolution. Nu för tiden så beställer vi alla sorters varor med hjälp av internet. Det är varor som finns runt om hela världen. Tillslut så kanske det inte finns några fysiska butiker kvar. Om internet från ingen stans skulle sluta fungera så hade förmodligen samhället rasat samman. Därför att vi människor styrs så mycket av det.
